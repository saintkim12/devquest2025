## Dev 퀘스트 2025-07-01: 하루 집중 TODO 앱

1. 오늘 날짜를 기준으로, 할 일을 여러 개 등록할 수 있어야 해요. ✅

- UI는 입력창, 할 일 목록, 체크박스, 삭제버튼 구성  
- 완료된 항목은 목록 아래로 이동  
- 오늘 날짜 상단 고정  
- 모바일에서도 간단한 반응형 대응

---
