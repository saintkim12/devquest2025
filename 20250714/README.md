## 🧩 오늘의 미션: **메모 리스트 필터링 기능 만들기**

### 🎯 요구사항 (2시간 안에 유의미한 결과 도출 목표)

1. 하단 리스트 영역에서 항목을 **검색창에 입력한 키워드로 필터링**할 수 있게 한다.
2. 필터는 **제목 또는 태그**에 포함된 텍스트가 있으면 해당 항목만 보여준다.
3. 입력이 비어 있으면 전체 항목이 다시 보인다.

---

## ✅ 오늘의 목표 기준

* 단순 JavaScript로 구현해도 되고, React + 상태로 구현해도 OK
* 최소 목표: 검색창 입력 시 리스트 필터링이 되는 것 확인
* 유의미한 결과:

  * "리스트 필터링 기능이 동작함"을 눈으로 확인 가능
  * 나중에 검색 외에 정렬, 태그 필터, 날짜 필터 등으로 확장 가능
